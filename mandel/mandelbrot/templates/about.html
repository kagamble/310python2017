The Mandelbrot Set

Take a point (real, imaginary) in the xy complex plane.

Set z = (x,y) = (real, imaginary).

Iterate the computation:

           z = z*z                         + (real, imaginary)
             = (x,y)*(x,y)                 + (real, imaginary)
             = (x+iy)*(x+iy)               + (real, imaginary)
             = x*x + 2ixy - y*y            + (real, imaginary)
             = (x*x - y*y, 2xy)            + (real, imaginary)

             = (x*x - y*y + real, 2xy + imaginary)

If z GROWS more than 2 units distant from (0,0)
   then the point (real, imaginary) is NOT in the Mandelbrot set
   and the corresponding pixel is colored based on the ITERATIONS needed to ESCAPE

If z STAYS within 2 units of (0,0)
   then the point (real, imaginary) is IN the Mandelbrot set
   and the corresponding pixel is colored BLACK
